vitePress快速搭建个人博客 ​

主要实现的功能有以下几个方面：

  • 1. 默认主题扩展
  • 2. 文档开发
  • 3. 搜索
  • 4. 评论
  • 5. 自动构建和部署
  • 5. 广告

项目初始化 ​

首先需要本地安装Node.js，需要16版本及以上。 推荐使用pnpm，命令行安装npm install -g pnpm。 创建工程，安装VitePress。首先创建文件夹，并打开命令行。

mkdir docs && cd docs pnpm init pnpm add -D vitepress pnpm add sass -D 安装sass pnpm dlx vitepress init 初始化
